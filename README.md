# Akshani's A/L Exam Tracker 🌸

A beautiful, interactive static website designed to help track A/L exam progress over a 35-day schedule. 

## Features
- **Modern UI:** Glassmorphism design with a cute pastel palette.
- **Dynamic Chart:** Real-time donut chart summarizing completion status.
- **Animations:** A fun confetti party animation plays whenever a day's tasks are completed.
- **Notes:** Ability to write and save small notes for every single day.
